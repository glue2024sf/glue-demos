SYSNAG｜SES・PMO 商談／人材／アサイン管理 Sales Cloud 画面デモ（叩き台・v0.1）
Sales Cloud SES/PMO レコードタイプ 案件⇔人材マッチング アサイン粗利自動計算 見積帳票(APEXボタン+Sander)
スプレッドシート二重入力＋i-SeiQ別入力を Salesforce に集約する画面デモ。案件（SES/PMO）のパイプライン（カンバン）を起点に、必要スキル×単価×稼働時期で人材候補をスコア提示（マッチング）し、アサイン（案件×人材）で粗利・粗利率を自動計算。ホーム（稼働率・待機・パイプライン）／案件カンバン・詳細／マッチング／タレント一覧・詳細／アサイン／取引先／見積帳票／経営ダッシュボードの10画面を業務フローで横断確認できます。見積帳票は「AppExchangeカスタムオブジェクトNG→APEXボタン＋Sanderライセンス」前提のモック。ヘッダで SES/PMO 事業を切り替え、右下トグルで各画面の「💡論点（要確認）」を表示できます。
※サンプルデータ（企業名・人材イニシャル・単価・粗利はすべてダミー）
イツモスマイル｜保育シフト自動配置 Service Cloud + Community デモ（叩き台・v0.1）
Service Cloud Community(利用者) 保育園シフト 自動配置(ルールベース) 配置基準・公休・時間集計
Excel＋LINE による保育園の職員シフト管理を Salesforce に集約する画面イメージ（叩き台）。管理者(Service Cloud)とスタッフ(Community・利用者)をヘッダで切り替えて確認できます。管理者側は「月間シフト表（記号×日付・基準違反を自動ハイライト）／自動シフト生成（早番2・遅番2・出勤6名・公休11日・資格/クラス・ルーティーンA→D→L→H を満たす配置をルールベースで生成し、満たせない日は埋めずにアラート）／職員台帳（資格・担当クラス・公休/有休/労働時間の自動集計）／希望・申請／配置基準の集計」。スタッフ側は「マイシフト・公休/有休残・希望休の提出」。手計算の時間集計（Excelでは #REF! が発生）を自動化し、特化型シフトツールとの違い＝職員・園児クラス・労務を同じ土台で一元管理、を体感できます。
※サンプルデータ（氏名・クラス・数値はすべてダミー。生成AIではなくルールベースの論理配置）
おしごとらぼ｜栄養士・調理師 求人サイト（Experience Cloud デモ・v0.1）
Experience Cloud 求人サイト AI履歴書・職務経歴書 閲覧履歴×掘り起こし Google for Jobs
栄養士・調理師に特化した人材紹介の自社求人サイトを Salesforce Experience Cloud で構築する画面デモ。求職者向けサイト（求人検索／求人詳細＋Google for Jobs／AIによる履歴書・職務経歴書作成／マイページの閲覧履歴）と、社内Salesforce（求職者レコードに行動が自動蓄積／休眠会員の掘り起こしリスト／求人管理／ダッシュボード）の2つの世界を切り替えて、サイトと顧客管理が一体になる価値を確認できます。
※サンプルデータ（施設名・求人・氏名・給与はすべてダミー）
日本システム｜営業管理 Sales Cloud 画面イメージ（叩き台・v1.1）
Sales Cloud 標準オブジェクト中心 医療/民事 レコードタイプ Outlook連携 簡易日報
Excel・日報による属人的な営業管理を Salesforce に集約する画面イメージ（叩き台）。取引先を起点に「商談・活動・導入システム」を一元可視化し、現場は「商談＋活動＋日報」を最小入力で回せる設計。ホーム／取引先詳細／商談（パス＋最小コア＋簡易見積）／活動記録／簡易日報の5画面を、各画面の「旧設計→新設計」の変更意図つきで確認できます。商談はレコードタイプで医療(B)／民事・既存(A)を分離、活動はOutlook（M365）と双方向同期、社内・移動の内部活動も記録可能。
※サンプルデータ（病院名・担当者・金額はダミー）
HIG｜ホテルアメニティ卸 Sales Cloud 画面モック (v0.2)
Sales Cloud 取引先（親子） 商談・粗利 サンプル追客
ホテルアメニティ卸の営業管理を Salesforce で一気通貫管理する画面モック。重点チェーン（取引先＝本社・施設の親子）／商談（提案明細・想定粗利・経由代理店）／活動（訪問・TEL・DM）／サンプル追客（フラグ・期限アラート）／商品・価格の業務画面に加え、ホームで新規・ロスト管理と集計の二重軸（自社年度・ブランド元ベース）切替を確認できます。
※サンプルデータ（企業名・担当者・金額・粗利はダミー）
AION｜営業 リード〜受注 Sales Cloud 画面デモ (v0.1)
Sales Cloud ChatWork×Claude クラウドサイン スプシ/Notion同期
営業のリード〜受注を Salesforce で一気通貫管理する画面デモ。紹介（4ステージ）／アウトバウンド（6ステージ）の経路別ステージ、ChatWork投稿をClaudeが解析して自動更新、既存スプレッドシート/Notionとの同期、見積〜クラウドサイン、モニタリング系・アラート系ダッシュボードの10画面を業務フローで横断確認できます。
※サンプルデータ（企業名・担当者・金額はダミー）
京都アソシエイツ｜Agentforce 対話型候補者検索 (v0.5)
Agentforce Data Cloud 自然言語検索
ハイクラス人材紹介向けの対話型候補者検索。一発で結果を出さず「件数＋絞り込み軸＋代表事例」を返し、200件→30〜50件→確定まで対話で絞り込みます。 5つの検索パターン（評価×年齢×ポジション／トピック×期間／複合スキル（地名辞書拡張）／HRポジション多層／地域×経歴×財務）を実装。右側パネルで抽出条件・件数推移・参照項目・裏ロジック（辞書/同義語）・生成クエリを可視化します。
